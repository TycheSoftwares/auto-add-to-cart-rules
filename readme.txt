Plugin is useful in WooCommerce to automatically add the product to the cart when different conditions are matched.

== Description ==

This plugin is very usefull for the WooCommerce store owners to automatically add the products to the cart based on the different scenarios when 

the particular conditions are matched the products selected for the particular scenarios will be added automatically to the cart..

Auto Add To Cart Rules enables to do the following things :

1. When the product of particular category is added to cart then it automatically adds the selected product to the cart.
2. When the total of cart reaches certain amount which is decided by the admin it automatically adds the selected product to the cart.
3. when any customer visits the website then the selected product will be added automatically to the cart.
4. All scenarios will work only to selected user role by the admin.
5. Admin can set the price of the automatically added product to 0.
6. Gives to function to automatically add the products only when the customer places their first order.
7. Also there is an option to not to add the product once it is removed the customer.

== Installation ==

Method 1:
<ol>
<li> Go to WordPress Dashboard->Plugins->Add New </li>
<li> Search Auto Add To Cart Rules plugin using search option </li>
<li> Find the plugin and click Install Now button </li>
<li> After installation, click on Activate Plugin link to activate the plugin. </li>
</ol>

Method 2:
<ol>
<li> Download the plugin auto-add-to-cart-rules.zip </li>
<li> Unpack the auto-add-to-cart-rules.zip file and extract the auto-add-to-cart-rules folder </li>
<li> Upload the plugin folder to your /wp-content/plugins/ directory </li>
<li> Go to WordPress dashboard, click on Plugins from the menu </li>
<li> Locate the auto-add-to-cart-rules and click on Activate link to activate the plugin. </li>
</ol>

Method 3:
<ol>
<li> Download the plugin auto-add-to-cart-rules.zip </li>
<li> Go to WordPress Dashboard-Plugins-Add New </li>
<li> Click on Upload Plugin link from top </li>
<li> Upload the downloaded auto-add-to-cart-rules.zip file and click on Install Now </li>
<li> After installation, click on Activate Plugin link to activate the plugin. </li>
</ol>

== Frequently asked questions ==

= Is multiple scenarios possible ? =

Yes. It is possible to select multiple scenarios.

= What happens if selected scenario have not selct the product to be added automatically ? =

Error will be shown on the cart page.

= Is there option to add product automatically to cart when cart total reached to some amount? =

Yes. There is an option available for that.

= Option available to add product automatically to cart based on different category product is added to cart? =

Yes. There is an option in which you can select different category.

= Can we apply rules only if the customer is logged in? =

Yes. There is an option to select the user role in which the functions will work only for that selected user.

= Product which will be automatically added to cart will be free? =

Its depends on the admin. There is an option given to make the price of automatically added product to 0.

= Is there an option to add product automatically to cart when customer visits website? =

Yes. There is an option to add product automatically to cart when customer visits website.
